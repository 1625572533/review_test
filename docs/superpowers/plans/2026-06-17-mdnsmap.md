# mdnsmap Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Go CLI that discovers mDNS-advertised assets in an IPv4 CIDR and port range, then prints services with deep banners.

**Architecture:** The CLI layer parses flags and delegates to small internal packages. `internal/mdns` handles DNS wire encoding and parsing, `internal/filter` handles CIDR and port matching, `internal/scan` sends multicast queries and aggregates responses into assets, and `internal/output` renders the text report.

**Tech Stack:** Go 1.22+ standard library only, `go test ./...`, UDP multicast to `224.0.0.251:5353`.

---
